### issue

issue 是原始问题的还原，包括 __题目标题__ 、__内容__、__解题示例(`Python3`)__、__问题类型(`label`)__

一个 issue 对应一个题，方便讨论，label 表示题的类别

脚本文件中也有冗余的 __原始问题__


### git commit message

git commit -m 的格式为 `"题目 | issue id | message"` 方便定位issue
